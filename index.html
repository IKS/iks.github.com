Interactive Knowledge Stack (IKS) is an open source community, whose projects are focused on building an open and flexible technology platform for semantically enhanced Content Management Systems (CMS). The software libraries hosted at our github repositories are focussed on interacting with semantic enriched content. They are built as widgets and applications on top of the VIE framework.

VIE - Vienna IKS editables

VIE, or Vienna IKS Editables is a project to make the content of your web pages editable through annotations. This means that by simply including some RDFa on your pages you can built rich user interactions and interfaces on the JavaScript level.

VIE widgets

  • hallo is a simple rich text editor (contentEditable) for jQuery UI.
  • Annotate is a is jQuery UI widget for annotating content via entity recognition back-end engines like Stanbol.
  • The image widget retrieves image from external repositories for RDFa encoded entities
  • The tagging widget
